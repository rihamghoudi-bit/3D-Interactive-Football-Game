# 3D Interactive Football Game

A 3D browser-based football game built with **Three.js** that integrates real-time **Hand Tracking** using MediaPipe to allow players to interact with the ball using hand gestures.

## 🚀 Features
- **3D Environment:** Immersive stadium view.
- **Hand Tracking:** Interaction using index finger positioning via webcam.
- **Real-time Feedback:** Updates player score and game state dynamically.

## 🛠 Tech Stack
- **Library:** Three.js
- **Computer Vision:** MediaPipe Hands
- **Environment:** Browser-based (JavaScript)

## 💻 How to Run
1. Clone the repository.
2. Open `index.html` in a modern web browser.
3. Grant camera permissions when prompted to enable hand tracking.
